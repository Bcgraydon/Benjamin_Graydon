Small Business Network - Packet Tracer Project

Overview

This project is a small business network built in Cisco Packet Tracer. The goal was to design a realistic, segmented network with basic security controls, then validate connectivity and document testing results.

Goals

  • Design a small business topology using switches, routers, and end devices.
  • Segment traffic using VLANs (separate departments and guest access).
  • Enable inter-VLAN routing and verify expected connectivity.
  • Apply ACL rules to restrict access between VLANs while still allowing required services.
  • Test with ping and basic troubleshooting commands to confirm the network meets requirements.

Topology and Components

  • Switching: VLANs configured on access ports, trunk links where needed.
  • Routing: Inter-VLAN routing (router-on-a-stick or Layer 3 interface) depending on the design.
  • Addressing: Separate subnet per VLAN with a default gateway per VLAN.
  • Security: ACLs to block unauthorized VLAN-to-VLAN traffic and isolate guests.

Testing and Validation

I validated the design by testing reachability within each VLAN and confirming that restricted traffic between VLANs was blocked as intended. Troubleshooting was done using basic router and switch commands and reviewing ACL matches.

  • Connectivity: Verified hosts can reach their default gateway and required internal resources.
  • Segmentation: Confirmed VLAN separation at Layer 2.
  • ACL policy: Confirmed blocked paths fail (expected) and allowed paths succeed (expected).
  • Troubleshooting: Used show commands (interfaces, VLANs, trunks, access-lists) to confirm configuration.

Skills Demonstrated

  • VLAN planning and switch port configuration (access vs trunk)
  • Subnetting and IP addressing
  • Inter-VLAN routing concepts
  • ACL design and verification (least privilege, guest isolation)
  • Network testing and troubleshooting using structured checks
